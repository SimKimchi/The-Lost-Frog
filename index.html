The Lost Frog

Controls

Movement:
  • A : Run left
  • D : Run right
  • Space : Jump
Wall Climb:
  • W : Climb up
  • S : Climb down
  • Space : Wall jump
Ceiling Climb:
  • A : Climb left
  • D : Climb right
  • S : Stop climbing
Actions:
  • Up : Attack up
  • Down : Attack down
  • Left : Attack left
  • Right : Attack right
